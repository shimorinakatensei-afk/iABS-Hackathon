# iABS — Учёт Аренды

Полнофункциональная банковская система учёта аренды на Python FastAPI.

## Описание

Внутренняя банковская система для управления договорами аренды с полной бизнес-логикой, аудитом и ролевым доступом.

## Технологии

- **Backend**: Python 3.8+, FastAPI
- **Frontend**: HTML, CSS, JavaScript (Vanilla)
- **База данных**: In-memory (легко заменяется на SQLite/PostgreSQL)
- **UI**: Тёмная тема в стиле банковских систем

## Установка и запуск

### 1. Установите зависимости

```bash
pip install -r requirements.txt
```

### 2. Запустите сервер

```bash
uvicorn main:app --reload
```

Приложение будет доступно по адресу: **http://localhost:8000**

## Основные модули

### 1. Сдача в аренду (Lease Out)
- Управление договорами сдачи в аренду
- Фильтры по статусу и поиск
- Действия: Добавить, Изменить, Удалить, Утвердить, Возврат, Протокол

### 2. Получение в аренду (Lease In)
- Управление договорами получения в аренду
- Аналогичный функционал

### 3. Справочники (References)
- Управление контрагентами (арендаторы/арендодатели)
- CRUD операции

### 4. Настройки (Settings)
- Доступно только для SUPERADMIN
- Управление ролями пользователей
- Просмотр матрицы прав доступа

## Бизнес-логика

### Машина состояний

```
DRAFT → APPROVED → RETURNED
```

**Правила:**
- Редактировать/удалять можно только DRAFT
- APPROVED заблокирован для изменений
- RETURNED - финальное состояние
- Нельзя утверждать свой договор

### Автоматическая генерация служебных записок

При утверждении генерируется номер: `MEMO-2026-XXXXX`

### Система аудита

Все действия логируются:
- Кто выполнил
- Что изменил
- Когда произошло
- Старое и новое значение

Просмотр через кнопку "Протокол".

## API Endpoints

### Аутентификация
- `POST /api/login` - Вход в систему
- `POST /api/logout` - Выход
- `GET /api/me` - Текущий пользователь

### Договоры
- `GET /api/contracts` - Список договоров (с фильтрами)
- `POST /api/contracts` - Создать договор
- `PUT /api/contracts/{id}` - Обновить договор
- `DELETE /api/contracts/{id}` - Удалить договор
- `POST /api/contracts/{id}/approve` - Утвердить договор
- `POST /api/contracts/{id}/return` - Вернуть из аренды

### Аудит
- `GET /api/audit/{entity_type}/{entity_id}` - Журнал аудита

### Справочники
- `GET /api/counterparties` - Список контрагентов
- `POST /api/counterparties` - Создать контрагента
- `DELETE /api/counterparties/{id}` - Удалить контрагента

### Настройки (только SUPERADMIN)
- `GET /api/users` - Список пользователей
- `PUT /api/users/{personnel_number}/role` - Изменить роль

## Переход на реальную БД

Для использования SQLite/PostgreSQL замените in-memory хранилище на SQLAlchemy:

```python
from sqlalchemy import create_engine
from sqlalchemy.orm import sessionmaker

engine = create_engine('sqlite:///iabs.db')
SessionLocal = sessionmaker(bind=engine)
```
